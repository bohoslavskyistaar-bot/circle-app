# Коло акордів

Перший MVP інтерактивного кола акордів.

## Що вже є

- SVG-коло квінт з мажорними, мінорними та зменшеними акордами.
- Поп-ап акорду після натискання сектора.
- Підсвічування клавіш на MIDI-клавіатурі.
- Програвання акорду через Web Audio.
- Адаптивний інтерфейс для десктопу та малого екрана.

## Запуск

Варіант без сервера: відкрий `index.html` у браузері.

Варіант із локальним сервером:

```bash
npm run start
```

Якщо PowerShell блокує `npm`, запусти:

```powershell
npm.cmd run start
```

Після цього відкрий `http://localhost:4173`.

## Наступний крок для desktop

Цю веб-основу можна загорнути в Tauri 2: тоді той самий UI піде на Windows, Linux і macOS, а пізніше стане базою для Android та iOS. Для цього треба буде додати Tauri-проєкт, іконки, назву застосунку, підписування білдів та CI-збірки.
